Tools

Digital tools and prototypes for archaeological research, spatial analysis, and lightweight visual workflows.

Tool philosophy

These tools are designed as lightweight, browser-based research instruments. Priority is given to clarity, portability, and accessible sharing over feature bloat.

Available

Spica

Browser-based 3D spatial visualisation tool for archaeological datasets.

  • Input: CSV datasets
  • Output: interactive visual exploration in browser
  • Status: stable
